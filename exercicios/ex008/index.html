Utilizando Tags Semânticas

Tag para Negrito

Para destacar algum texto em negrito, é utilizado a tag ‹strong›

Tag para Itálico

Para destacar algum texto em Itálico, é utilizado a tag ‹em›

Tag para Marcar Texto

Para destacar algum texto com o Marca Texto, é utilizado a tag ‹mark›

Tag para Texto Deletado

Para utilizar a tag de Texto Deletado, é utilizado a tag ‹del›

Tag para Texto Inserido (Sublinhado)

Para utilizar a tag de Texto Inserido ou Sublinhado, é utilizado a tag ‹ins›

Tag para texto sobrescrito e texto subescrito

Para utilizar a tag sobrescrito ou subescrito, são utilizadas as Tags de ‹sup› ‹sub›

Tag para Texto monoespaçado

Para deixar o texto monoespaçado, é utilizada a tag ‹code›

Tag para utilizar citações

Para utilizar citações, utilize a Tag ‹q›. Dessa forma, será feita uma citação simples

Tag para utilizar citações compostas Em Blocos

A tag ‹blockquote› é utilizada para criar citações completas, onde pode ser definido um parâmetro cite="", que pode ser adicionado o link da determinada citação.

De acordo com o autor Jeff Nobbles, em seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto.
